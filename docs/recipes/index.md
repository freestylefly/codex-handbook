# 实战案例库

这里会收集可复现、可改写、可复制到真实项目里的 Codex 使用案例。

案例库分三层：

- 官方能力启发案例：从 OpenAI 官方文档、产品页、Help Center 和公开博客中提炼。
- 社区通用实践案例：来自开源项目、issue/PR 工作流和开发者常见任务。
- 个人实测案例：后续由维护者和贡献者补充完整过程、截图和复盘。

## 案例格式

每个案例建议包含：

- 场景：什么项目、什么问题。
- 目标：希望 Codex 完成什么。
- 任务说明：可以直接复制的 prompt。
- 过程要点：Codex 应该先读什么、跑什么。
- 验证方式：命令、测试、人工检查。
- 风险提醒：什么时候不适合直接照抄。
