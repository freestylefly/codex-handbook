## 仓库定位

这是一份面向中文开发者的 Codex 教程知识库。它围绕真实工程流程重组官方资料、工程经验和中文开发者常见问题，帮助你从上手走向稳定交付。

![Codex 使用入口地图](/images/codex-surfaces.svg)

你可以把它当成四样东西：

- 新手入门手册：入口选择、安装、登录、第一条任务、常见错误。
- 工程实践手册：如何让 Codex 更稳定地读项目、改代码、跑测试和提交 PR。
- 团队知识库：如何写 `AGENTS.md`、沉淀案例、建立安全边界和协作规范。
- 非开发工作台：如何做文档、学习、发布说明、知识整理和流程自动化。

## 建议阅读顺序

1. 先读 [学习路线](/guide/00-overview.html) 和 [入口地图](/platform/)。
2. 按顺序完成 [安装与登录](/guide/01-installation.html) 和 [第一次改代码](/guide/02-first-run.html)。
3. 用 [任务设计](/practice/task-design.html) 改写自己的真实需求。
4. 遇到具体任务时去 [实战案例库](/recipes/) 找模板。
5. 如果要在团队内推广，优先补齐 [AGENTS.md](/guide/05-agents-md.html) 和 [沙盒与审批](/guide/06-sandbox-approvals.html)。
