## 仓库定位

这是一份面向中文开发者的 Codex 教程知识库。它不是官方文档的翻译，也不是零散技巧合集，而是围绕真实工程流程重组后的学习材料。

你可以把它当成三样东西：

- 新手入门手册：安装、登录、第一条任务、常见错误。
- 工程实践手册：如何让 Codex 更稳定地读项目、改代码、跑测试和提交 PR。
- 团队知识库：如何写 `AGENTS.md`、沉淀案例、建立安全边界和协作规范。

## 建议阅读顺序

1. 先读 [学习路线](/guide/00-overview.html)，确定你现在处在哪个阶段。
2. 按顺序完成 [安装与登录](/guide/01-installation.html) 和 [第一次改代码](/guide/02-first-run.html)。
3. 遇到具体任务时去 [实战案例库](/recipes/) 找模板。
4. 如果要在团队内推广，优先补齐 [AGENTS.md](/guide/05-agents-md.html) 和 [沙盒与审批](/guide/06-sandbox-approvals.html)。
